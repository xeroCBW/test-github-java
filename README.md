# test-github-java
test-github-java 使用eclispe 直接上传仓库,不需要先创建仓库再去项目

注意点:

1. 创建新的本地仓库

2. 强制提交

3. 需要在github中创建仓库，不能像sourceTree 一样，没有帮你创建仓库
